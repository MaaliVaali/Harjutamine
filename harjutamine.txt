GitHub on Ameerika firma, mis pakub veebimajutusteenust tarkvaraarenduseks, kasutades Giti tarkvara.
GitHubi teenus on suurelt jaolt tasuta, kuid GitHub pakub tasu eest teenust IT-professionaalidele, töörühmadele ja ettevõtetele.
GitHubi tasuta teenust kasutatakse tihtipeale avatud lähtekoodiga tarkvara arenduseks.
Saan allalaadida erinevaid koode, samuti saan ise koode üleslaadida.
Lisaks sellele võimaldab GitHub ka:
koodi dokumentatsiooni README-failide kaudu
vigadest teatamist (kaasa arvatud soovitusi koodi parandamiseks ja uuteks funktsioonideks)
projektidele viki loomist
võimalust ise koodi parandada ning pakkuda selle kasutamist koodivaramu haldaja(te)le ("pull request")
muudatuste ajalugu
graafikuid ja muud statistikat iga projekti kohta
